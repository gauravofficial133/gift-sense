## Template 1: "Gilded Bloom"

### Occasion Type
Anniversary, Valentine's Day, romantic milestones

### Creative Concept / Emotional Theme
**"Love letters pressed between pages of time."**

The card evokes the feeling of opening an old love letter found inside a book of dried flowers. It merges botanical illustration with aged-paper warmth, creating an artifact that feels like it already has sentimental value before the recipient even reads it. The tension between structured elegance (the gold frame) and organic softness (the sprawling botanicals) mirrors how relationships balance commitment with spontaneity.

### Detailed Layout Structure

**Canvas:** Portrait A5 (420 x 595)

```
+-----------------------------------------------+
|  [Outer gold hairline frame, 8px inset]       |
|                                                |
|    [Botanical spray — top-left corner]         |
|     Peony + eucalyptus + small berries         |
|     Spills diagonally ~35% into the card       |
|                                                |
|              [Headline zone]                   |
|         Positioned at 38% from top             |
|         Right-aligned to center axis           |
|         Script font, large (32-36px)           |
|         Gold/deep rose color                   |
|                                                |
|         [Thin ornamental rule — 60px]          |
|         Centered, filigree style               |
|                                                |
|              [Body message zone]               |
|         Centered, max-width 75%                |
|         Serif font, 14-15px                    |
|         3-4 lines, generous line-height (1.8)  |
|                                                |
|         [Closing + signature]                  |
|         Right-aligned                          |
|         Italic serif, muted tone               |
|                                                |
|    [Botanical spray — bottom-right corner]     |
|     Mirrored/complementary arrangement         |
|     ~25% coverage, lighter opacity (0.7)       |
|                                                |
|  [Inner decorative frame — double line, 16px]  |
|  [Subtle aged-paper texture overlay, 0.04 op]  |
+-----------------------------------------------+
```

**Layer order (back to front):**
1. Warm cream background (#FDF6EC)
2. Subtle paper-grain texture (noise pattern at 0.04 opacity)
3. Inner double-line frame (muted gold)
4. Outer hairline frame (deeper gold)
5. Bottom-right botanical spray (0.7 opacity)
6. Body text + closing + signature
7. Ornamental rule divider
8. Headline text
9. Top-left botanical spray (full opacity)
10. Small scattered petals (2-3 loose petals floating between botanicals)

### Typography Style

**Heading:** Flowing, confident script with natural stroke variation. Mood: a handwritten love letter from someone with beautiful penmanship. Think Great Vibes or Dancing Script weight. Not overly decorative — legible at a glance, but unmistakably personal.

**Body:** Warm, literary serif. Mood: the inside page of a poetry book. Think Cormorant Garamond or Source Serif Pro. Slightly condensed letter-spacing, generous line-height. The text should breathe, not crowd.

### Color Palette

| Role | Color | Hex | Reasoning |
|------|-------|-----|-----------|
| Background | Antique cream | `#FDF6EC` | Warm, paper-like, non-clinical white |
| Primary | Burnished rose | `#8B3A5C` | Deep enough for contrast, romantic without being juvenile pink |
| Accent | Soft gold | `#C9A96E` | Metallic warmth, elevates perceived value, suggests "gilded" |
| Ink | Deep espresso | `#3B2F2F` | Warmer than pure black, feels handwritten |
| Muted | Dusty mauve | `#9C8B8B` | Recedes gracefully for closing/signature text |

**Why this palette works:** The cream-rose-gold triad is the color language of luxury stationery. It signals "this was chosen with care" before anyone reads a word. The espresso ink avoids the coldness of black, keeping everything intimate.

### SVG / Illustration Requirements

**Botanical Elements:**
- `peony_bloom_full.svg` — Large open peony, front-facing, 5-6 layered petals with visible stamen center
- `peony_bud.svg` — Closed/half-open peony bud with sepals
- `eucalyptus_branch.svg` — 8-10 leaf pairs on a gently arcing stem
- `small_berries_cluster.svg` — Cluster of 5-7 round berries on thin branching stems
- `loose_petal_1.svg` — Single fallen petal, slightly curled
- `loose_petal_2.svg` — Another single petal, different orientation
- `tiny_leaves_spray.svg` — Small filler foliage, 4-5 tiny leaves

**Decorative Assets:**
- `filigree_rule.svg` — Ornamental horizontal divider, ~60px wide, symmetrical scroll pattern
- `gold_frame_corner.svg` — Small decorative corner piece for the inner frame (used 4x, rotated)
- `paper_noise_tile.svg` — Tileable subtle grain pattern for the aged-paper texture layer

**Background Elements:**
- None beyond the solid fill + texture overlay. The botanicals ARE the visual interest.

### Why Users Would Pay for It

This card occupies a space that digital greetings almost never reach: **it looks like something you'd frame**. The botanical-and-gold combination is the single best-selling aesthetic on Etsy for anniversary/love cards (consistently 4.8+ star ratings across thousands of shops). But here, the *message is personalized from real conversations* — something no Etsy card can offer. The visual quality provides the "worth printing" threshold; the AI-written copy provides the "this is actually about us" emotional hit. Together, they justify the user choosing this over a $6 Hallmark card or a $3 Canva template they'd have to write themselves.

## Template 2: "Confetti Cinema"

### Occasion Type
Birthday, celebration, promotion, graduation

### Creative Concept / Emotional Theme
**"The moment right after the surprise, when everyone's laughing."**

This card captures peak celebration energy — not the formal, posed-photo version, but the candid, mid-laugh, confetti-still-falling moment. It uses bold geometric confetti, a dramatic dark-to-bright gradient, and punchy display typography to create something that feels like a party invitation designed by a studio, not a greeting card. The asymmetric composition prevents it from feeling stiff; the negative space prevents it from feeling chaotic.

### Detailed Layout Structure

**Canvas:** Landscape A5 (595 x 420)

```
+---------------------------------------------------------------+
|  [Gradient background: deep navy → midnight blue]             |
|                                                                |
|  [Confetti burst — top-right quadrant]                        |
|   Geometric shapes: circles, rectangles, diamonds             |
|   Mix of gold, coral, electric blue, white                    |
|   Varying sizes (4px to 16px), scattered with rotation        |
|   Density: thick at top-right, thinning toward center         |
|                                                                |
|  [LEFT 55% of card — text zone]                               |
|                                                                |
|      [Headline — positioned at 30% from top]                  |
|      Left-aligned, flush with 12% left margin                 |
|      Display font, LARGE (40-48px)                            |
|      White or bright gold                                     |
|      Can be 2 lines if needed                                 |
|                                                                |
|      [Accent stripe — 3px, coral, 40% width]                  |
|      Left-aligned under headline                              |
|                                                                |
|      [Body message]                                           |
|      Left-aligned, max-width 50% of card                      |
|      Sans-serif, 14px, white at 0.85 opacity                  |
|      2-3 lines, tight line-height (1.5)                       |
|                                                                |
|      [Closing]                                                 |
|      Left-aligned, small caps, 11px, gold                     |
|                                                                |
|      [Signature]                                               |
|      Left-aligned, italic, 13px, white at 0.6                 |
|                                                                |
|  [RIGHT 45% — illustration zone + confetti continues]         |
|                                                                |
|  [Bottom-left: 2-3 scattered confetti pieces, sparse]         |
|  [Bottom-right: subtle star burst, low opacity]               |
|                                                                |
+---------------------------------------------------------------+
```

**Layer order (back to front):**
1. Gradient background (linear, top-left navy to bottom-right dark blue)
2. Subtle radial glow from top-right corner (warm gold, 0.08 opacity, large radius)
3. Background confetti layer (large, low-opacity pieces, blurred feel)
4. Bottom-left sparse confetti
5. Star burst accent (bottom-right, 0.12 opacity)
6. Mid-ground confetti (medium pieces, medium opacity)
7. Text block (headline, stripe, body, closing, signature)
8. Foreground confetti (small, bright, sharp — over everything including text edges)
9. Single gold star or sparkle element near the headline (draws the eye)

### Typography Style

**Heading:** Bold, confident, wide-set display type. Mood: a movie poster title or a music festival headliner. Think Abril Fatface or Lobster — something that owns the space and demands attention. All caps or title case depending on the headline length. Slight letter-spacing (0.02em) for breathing room.

**Body:** Clean geometric sans-serif. Mood: the supporting text on a Spotify Wrapped card. Think Quicksand or Nunito. Light weight (300-400), comfortable at small sizes against the dark background. White with slight transparency so it supports rather than competes with the headline.

### Color Palette

| Role | Color | Hex | Reasoning |
|------|-------|-----|-----------|
| Background | Deep navy | `#0F1B2D` | Cinematic depth, makes bright elements pop dramatically |
| Background gradient end | Midnight blue | `#162447` | Subtle gradient prevents flatness |
| Primary | Bright gold | `#FFD166` | Celebration signal, high contrast on dark, luxury association |
| Accent | Living coral | `#FF6B6B` | Energy, warmth, trend-forward, excellent on dark backgrounds |
| Ink | Pure white | `#FFFFFF` | Maximum readability on dark, clean and modern |
| Muted | Soft white | `#FFFFFF` at 0.6 opacity | Recedes naturally on the dark background |
| Confetti extra | Electric blue | `#4ECDC4` | Third confetti color for visual richness |

**Why this palette works:** Dark backgrounds are the single most effective way to make a digital card look premium — it's why every streaming platform, music app, and film studio uses dark UI for their showcase moments. The gold-coral-teal confetti triad against navy creates a celebration that feels expensive, not childish. This is a birthday card for adults.

### SVG / Illustration Requirements

**Confetti Elements (individual pieces, composited at random positions):**
- `confetti_circle.svg` — Simple filled circle (used at multiple scales: 4px, 8px, 12px)
- `confetti_rect.svg` — Small rounded rectangle (used with random rotation)
- `confetti_diamond.svg` — Rotated square / diamond shape
- `confetti_star_4pt.svg` — Four-pointed star/sparkle
- `confetti_squiggle.svg` — Short wavy line segment (adds organic movement to geometric shapes)
- `confetti_ring.svg` — Circle outline only (stroke, no fill) — adds depth variation

**Decorative Assets:**
- `accent_stripe.svg` — Short horizontal line with soft rounded caps (3px height, variable width)
- `sparkle_cluster.svg` — 3-4 tiny four-pointed stars grouped tightly (placed near headline)
- `radial_glow.svg` — Large soft radial gradient circle (gold center → transparent edge, used as background glow)

**Background Elements:**
- `gradient_bg.svg` — Linear gradient from `#0F1B2D` to `#162447` (or rendered as CSS in the HTML template)
- `starburst_subtle.svg` — Radiating lines from a center point (low opacity, decorative, bottom-right)

### Why Users Would Pay for It

Birthday cards are the highest-volume greeting card category globally, yet most digital birthday cards look either childish (clip-art balloons) or sterile (plain text on white). This template solves both problems: the dark cinematic palette signals sophistication, the confetti geometry signals celebration, and the asymmetric layout signals "designed by a studio." On platforms like Canva, dark-themed celebration templates consistently outperform light ones for adult birthdays (25-45 age demographic). The landscape orientation also makes it optimized for screen sharing — the most common way digital cards are actually delivered (WhatsApp, Instagram Stories). The left-aligned text is a deliberate departure from the centered-text greeting card norm; it creates a magazine-editorial feel that makes the message feel more like a statement than a generic wish.

## Template 3: "Sacred Geometry"

### Occasion Type
Diwali, Raksha Bandhan, Eid, Holi, festive/cultural occasions, also works beautifully for formal thank-you and farewell

### Creative Concept / Emotional Theme
**"The quiet reverence before the festival begins."**

This card captures the meditative stillness that precedes celebration — the moment diyas are being lit, the rangoli is freshly completed, the house smells of incense and sweets. Rather than depicting specific religious symbols (which would limit its cross-cultural appeal), it uses sacred geometry and mandala-inspired patterns that are universal across South Asian, Middle Eastern, and global spiritual aesthetics. The result is something that feels deeply cultural without being exclusionary — elegant enough for a corporate Diwali greeting and personal enough for a family card.

### Detailed Layout Structure

**Canvas:** Square (500 x 500)

```
+-----------------------------------------------+
|                                                |
|  [Full background: deep jewel tone]           |
|                                                |
|  [Mandala ring — centered, 85% of card width] |
|   Geometric circular pattern                   |
|   Outer ring: repeating petal/arch motif       |
|   Middle ring: diamond lattice pattern         |
|   Inner ring: simple radiating lines           |
|   Center: open circle (text lives here)        |
|   Opacity: 0.12 — visible but not competing   |
|                                                |
|  [Corner ornaments — all 4 corners]           |
|   Quarter-circle filigree fans                 |
|   Gold on dark, 0.25 opacity                   |
|   Mirrored/rotated for each corner             |
|   Size: ~60x60px each                          |
|                                                |
|  [TEXT BLOCK — dead center of card]           |
|                                                |
|      [Headline — vertically centered at 38%]   |
|      Center-aligned                            |
|      Serif display, 28-32px                    |
|      Gold or warm white                        |
|                                                |
|      [Small diamond ornament, 8px]             |
|                                                |
|      [Body message]                            |
|      Center-aligned, max-width 70%             |
|      Light serif, 13-14px                      |
|      Warm white at 0.9 opacity                 |
|      2-3 lines                                 |
|                                                |
|      [Small diamond ornament, 8px]             |
|                                                |
|      [Closing + signature, stacked]            |
|      Center-aligned, 12px, gold at 0.7         |
|                                                |
|  [Scattered light dots — simulating diyas]     |
|   6-8 tiny glowing circles, random positions   |
|   Warm gold, varying opacity (0.15-0.35)       |
|   Soft glow effect (radial gradient per dot)   |
|                                                |
|  [Fine inner border — 20px inset]             |
|   Single line, gold, 0.2 opacity               |
|   Subtle corner notches (small squares)        |
|                                                |
+-----------------------------------------------+
```

**Layer order (back to front):**
1. Solid deep background (#1A1A2E royal indigo)
2. Mandala ring pattern (0.12 opacity, gold strokes on dark)
3. Diya glow dots — background layer (largest, most diffused, 0.08 opacity)
4. Inner border line
5. Corner ornament filigrees (4x)
6. Diya glow dots — mid layer (medium, 0.2 opacity)
7. Body text + ornamental diamonds
8. Closing + signature
9. Headline text (brightest element on the card)
10. Diya glow dots — foreground layer (smallest, sharpest, 0.35 opacity)

### Typography Style

**Heading:** Stately, high-contrast serif with visible thick-thin stroke variation. Mood: the title page of a hand-bound book, or gold embossing on a leather journal. Think Playfair Display or Cormorant Garamond at display weight. The letterforms themselves should feel like architecture — each character a small monument. Title case, moderate letter-spacing (0.03em).

**Body:** Gentle, readable serif with warmth. Mood: an invitation printed on handmade paper. Think Source Serif Pro or Merriweather at light weight. Slightly larger line-height (1.9) to let each line sit in its own space. The reader should feel unhurried.

### Color Palette

| Role | Color | Hex | Reasoning |
|------|-------|-----|-----------|
| Background | Royal indigo | `#1A1A2E` | Deep, meditative, associated with evening festivals and formal occasions across cultures |
| Primary | Warm gold | `#D4AF37` | The single most universal signifier of festivity, prosperity, and reverence |
| Accent | Amber glow | `#E8A849` | Warmer, softer gold for secondary elements — evokes diya flame light |
| Ink | Warm white | `#F5F0E8` | Softer than pure white, feels like candlelight on paper |
| Muted | Pale gold | `#D4AF37` at 0.5 opacity | Recedes while staying within the gold family |

**Why this palette works:** Indigo-and-gold is the definitive "festive formal" combination across South Asian design. It's used in wedding invitations, Diwali packaging, and premium ethnic wear branding. But it also reads as "elegant occasion" globally — it works for a retirement dinner or a New Year's greeting just as well. The single-family gold palette (instead of multi-color) keeps it sophisticated; the warm white ink ensures readability without introducing coldness.

### SVG / Illustration Requirements

**Mandala Elements:**
- `mandala_outer_ring.svg` — Repeating arch/petal pattern arranged in a circle (24-32 repeats). Stroke only, no fill. Sized to ~400px diameter on a 500px card.
- `mandala_middle_ring.svg` — Diamond/rhombus lattice pattern in a smaller concentric circle (~280px diameter)
- `mandala_inner_ring.svg` — Simple radiating lines from center (16 lines, thin stroke), ~160px diameter
- `mandala_center_void.svg` — Empty circle with a thin decorative border (this is where text goes — not filled)

**Corner Ornaments:**
- `corner_filigree.svg` — Quarter-circle ornamental fan with layered arcs and small dot accents. Designed for top-left; rotated 90/180/270 for other corners. ~60x60px.

**Decorative Assets:**
- `diamond_ornament_small.svg` — Tiny decorative diamond/rhombus (8px) used as section divider between text blocks
- `diya_glow.svg` — Small circle with radial gradient (bright center → transparent edge). Used at multiple scales for the scattered glow-dot effect. Not a literal diya shape — just the light it casts.
- `border_corner_notch.svg` — Small square or diamond cutout for the inner border corners (4x, rotated)

**Background Elements:**
- No additional background elements. The mandala IS the background treatment. Keeping the background simple is what makes the gold elements read as premium rather than busy.

### Why Users Would Pay for It

The Indian gifting market is massively underserved by digital card platforms. Canva has generic "Diwali" templates with clip-art diyas; Hallmark has nothing culturally relevant. There is genuine demand — evidenced by the explosion of WhatsApp forward-culture during Indian festivals — but almost zero supply of *premium, personalized, printable* festival cards that don't look like a forwarded JPEG from 2015. This template fills that gap: it's culturally resonant without being cliched (no cartoonish diyas, no clip-art rangoli), it's dark-and-gold which photographs beautifully for social sharing, and the square format is optimized for Instagram and WhatsApp stories. For GiftSense's India-focused market, this is the highest-conversion template because it serves an occasion type that competitors ignore while delivering visual quality they can't match.

### Asset Sourcing Recommendations

**Best free sources for these specific assets:**

- **Botanicals (#1-7):** SVGRepo "botanical line art" or unDraw floral. Alternatively, trace from Unsplash botanical photography. These need a consistent hand-drawn line-art style — mixing styles will break the template.
- **Confetti (#11-16):** Generate programmatically in Go. These are simple geometric shapes — they're better generated as inline SVG with randomized positions/rotations/colors than sourced as static files.
- **Mandala patterns (#21-24):** SVGRepo "mandala geometric" or generate programmatically with circular repetition functions. The repeating nature of mandalas makes them ideal for algorithmic generation.
- **Decorative assets (#8-9, 17-18, 25-26, 28):** SVGRepo "ornamental divider" and "filigree corner." These are the most common SVG categories and have thousands of free options.
- **Effects (#10, 19-20, 27):** Generate as inline SVG. Radial gradients, noise patterns, and radial lines are trivial in SVG markup and don't need external files.

**Assets that should be generated in code (not sourced):** #10, #11-16, #17, #19, #20, #23, #27 (13 of 28 — nearly half can be generated programmatically, reducing the sourcing burden to 15 unique illustration/decorative assets).
